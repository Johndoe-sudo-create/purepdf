## 🧾 What is purepdf?

purepdf is a tool that helps you create and edit PDF files. You can generate new PDFs, combine multiple PDFs into one, split PDFs into smaller parts, extract text, and even convert simple HTML pages into PDF documents. It runs on many platforms but this guide focuses on Windows.

You do not need any other software or special setups to use purepdf. It works alone, with no extra parts to install.

## 💡 What purepdf can do for you

- Create PDF documents with text, images, tables, and shapes.

- Combine several PDF files quickly.

- Get plain text out of PDF files.
